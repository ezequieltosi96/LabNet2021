Nortwind Traders

Frontend app for the '.NET Northwind webAPI' made with Angular v12

In this preliminary version you can perfom all the CRUD operations on the 'Category' entity.

The angular client must run on http://localhost:4200 and the web API must run on http://localhost:60168 If you want, you can change that on categoryService (angular) and on CategoryController (.NET)
